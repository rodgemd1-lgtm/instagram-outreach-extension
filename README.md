# Alex Recruiting

Production recruiting intelligence system for Jacob Rodgers.

Main app:
- `alex-recruiting/`

Production URLs:
- `https://alex-recruiting.vercel.app`
- `https://alex-recruiting.vercel.app/recruit`

Local development:

```bash
cd alex-recruiting
npm install
npm run dev
```

Useful commands:

```bash
cd alex-recruiting
npm run lint
npm test
npm run build
```

Notes:
- The Git repository root is this folder.
- The active Next.js app and all current product work live under `alex-recruiting/`.
- Legacy extension-era files remain at the repo root, but the recruiting product is the nested app.
